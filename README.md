# Protocolos de Rehabilitacion

Guias de rehabilitacion postquirurgica por fase y semana. Selecciona patologia y fecha de cirugia para ver las recomendaciones de la fase actual.

## Acceso directo
[Abrir la aplicacion](https://jmacot.github.io/rehabilitacion-cot/)

## Funcionalidades
- 64 protocolos organizados por region anatomica (rodilla, hombro, cadera, pie/tobillo, columna, codo, mano)
- Calculo automatico de fase segun fecha de cirugia o semanas postoperatorias
- Timeline visual D3.js con barra de progreso e indicador de fase actual
- Gantt de fases con semanas y marcador temporal
- Contenido clinico: objetivos, precauciones, carga de peso, intervenciones por categoria, criterios de progresion
- Videos de ejercicios enlazados a YouTube en 15 protocolos
- Filtro por region, busqueda por nombre de patologia
- Copiar fase al portapapeles
- Modo oscuro con auto-deteccion
- Responsive para movil y escritorio
